Web Dev?

what is web development?

Web development means designing a website or developing a pre built site. There are three types of development.

  1. Front-End development
  2. Back-end development
  3. Full Stack development

1:Front-End Development

Front-end is rendered by the browser what user usually see. You are reading content of a web page. This is front end. Front-end development mainly requires three language or skills. They are:

  1. HTML(Hyper Text Markup Language)
  2. CSS(Cascading Style Sheet)
  3. JavaScript

2:Back-End Development

Back-end means the server side of a website. Hendling request, storing data, sequrity etc. is done by the back-end of the website. Back-end developmant can.be done by many programming language. But the most used and popular language are:

  1. Python
  2. PHP
  3. JavaScript
  4. Ruby
  5. SQL
  6. Java

3:Full Stack Development

Full stack development is developing both the front-end and back-end of a web site. For becpming a full stack developer you need to know both front-end and back-end development. You should know at least Html, Css, JavaScript and one pf the back-end language.
